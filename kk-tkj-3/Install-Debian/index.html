INSTALL DEBIAN

Tampilkan Penjelasan:

Bagian 1: Mengunduh Debian

  1. Untuk memulai, buka browser favorit Anda dan kunjungi situs resmi Debian di https://www.debian.org/. Di situs ini, Anda dapat mengunduh ISO file Debian yang nantinya akan digunakan untuk instalasi pada perangkat Anda.

Bagian 2: Membuat Mesin Virtual di VirtualBox

  1. Setelah mengunduh Debian, langkah berikutnya adalah membuka VirtualBox. Pada jendela utama VirtualBox, klik tombol "New" untuk membuat mesin virtual baru. Ini akan menjadi tempat di mana Anda akan menginstal Debian.

  2. Setelah menekan tombol "New", Anda akan diminta untuk mengkonfigurasi mesin virtual baru. Ikuti langkah-langkah berikut:

    • Nama dan Folder: Masukkan nama untuk mesin virtual Anda, misalnya "Debian". Pilih lokasi folder tempat Anda ingin menyimpan mesin virtual ini.

    • ISO Image: Pilih file ISO Debian yang telah Anda unduh sebelumnya dengan menekan "Choose a disk file..." dan navigasikan ke lokasi file tersebut.

    • Unattended Installation: Centang kotak "Skip Unattended Installation" untuk menghindari instalasi otomatis, sehingga Anda dapat mengatur Debian sesuai preferensi Anda.

  3. Setelah memasukkan ISO image, Anda perlu mengatur Base Memory dan Processors. Sesuaikan jumlah memori (RAM) dan prosesor sesuai dengan kapasitas perangkat Anda. Pastikan untuk memberikan cukup memori dan prosesor agar Debian dapat berjalan dengan lancar di mesin virtual.

  4. Setelah mengatur memori dan prosesor, pilih opsi "Create a virtual hard disk now". Pada bagian ini, pastikan Anda memberikan ruang penyimpanan yang cukup agar Debian dapat beroperasi dengan baik. Disarankan untuk memberikan minimal 10-20 GB tergantung pada kebutuhan Anda.

  5. Setelah selesai mengatur virtual hard disk, klik tombol "Finish" untuk menyelesaikan pembuatan mesin virtual. Mesin virtual Anda sekarang sudah siap untuk instalasi Debian.

Langkah 3: Memulai Instalasi Debian

  1. Pilih mesin virtual yang tadi Anda buat dari daftar di VirtualBox, lalu klik tombol "Start" untuk memulai instalasi Debian. Mesin virtual akan mulai berjalan, dan Anda akan diarahkan ke proses instalasi.

  2. Setelah mesin virtual dimulai, Anda akan melihat layar boot Debian. Pada layar ini, pilih opsi "Install" untuk memulai proses instalasi Debian di mesin virtual Anda.

  3. Setelah memilih opsi "Install", Anda akan diminta untuk memilih bahasa. Pilih English sebagai bahasa yang akan digunakan selama proses instalasi Debian.

  4. Setelah memilih bahasa, Anda akan diminta untuk memilih lokasi. Pada menu lokasi, pilih opsi "Other" untuk memilih lokasi di luar opsi default yang ditampilkan.

  5. Setelah memilih "Other" pada menu lokasi, pilih "Asia" sebagai benua tempat Anda berada atau sesuai dengan lokasi yang Anda inginkan untuk konfigurasi sistem.

  6. Setelah memilih "Asia", pilih "Indonesia" sebagai negara Anda. Ini akan mengatur konfigurasi sistem sesuai dengan lokasi geografis Anda.

  7. Pada menu "Configure Locales", pilih "United States". Ini akan mengatur locale sistem berdasarkan pengaturan regional yang umum digunakan di Amerika Serikat

  8. Pada menu "Configure The Keyboard", pilih "American English" sebagai layout keyboard. Ini akan mengatur tata letak keyboard sesuai dengan standar bahasa Inggris Amerika.

  9. Pada menu "Configure the Network", masukkan "debian" sebagai nama host untuk mesin virtual Anda. Nama host ini akan digunakan untuk mengidentifikasi mesin virtual di jaringan.

  10. Pada menu "Configure the Network", biarkan field "Domain name" kosong. Anda tidak perlu mengisi informasi domain name jika tidak ada kebutuhan khusus untuk konfigurasi domain.

  11. Pada menu "Set up users and passwords", masukkan password untuk root sesuai dengan keinginan Anda. Pastikan password yang Anda pilih kuat dan mudah diingat, karena ini akan digunakan untuk akses administratif pada sistem Debian.

  12. Setelah memasukkan password untuk root, Anda akan diminta untuk "Re-enter password to verify". Masukkan kembali password yang sama untuk memastikan tidak ada kesalahan dalam penulisan.

  13. Pada menu "Setup users and passwords", masukkan nama lengkap Anda di kolom "Full name for the new user". Ini akan digunakan untuk membuat akun pengguna dengan nama yang sesuai dengan identitas Anda.

  14. Pada menu "Setup users and passwords", masukkan username yang diinginkan di kolom "Username for your password". Username ini akan digunakan untuk login dan mengakses sistem sebagai pengguna baru.

  15. Pada menu "Setup users and passwords", pilih "Choose a password for the new user". Masukkan password yang kuat dan aman untuk akun pengguna baru Anda. Pastikan untuk mengingat password ini, karena akan digunakan untuk login ke sistem.

  16. Setelah memilih password untuk pengguna baru, Anda akan diminta untuk "Re-enter password to verify". Masukkan kembali password yang sama untuk memastikan tidak ada kesalahan dalam penulisan.

  17. Pada menu "Configure the clock", pilih "Western" sebagai zona waktu Anda. Ini akan mengatur waktu sistem sesuai dengan wilayah waktu yang sesuai.

  18. Pada menu "Partition disks", pilih opsi "Guided - use entire disk". Ini akan memandu Anda untuk mempartisi seluruh disk yang tersedia untuk instalasi Debian, sehingga memudahkan proses instalasi dengan pengaturan partisi otomatis.

  19. Pada menu "Partition disks", pilih hard disk yang ada di komputer Anda. Ini adalah disk tempat Debian akan diinstal. Pastikan Anda memilih disk yang benar untuk menghindari kehilangan data dari disk lain.

  20. Setelah memilih hard disk, pada menu "Partition disks", pilih opsi "All files in one partition". Ini akan mengatur sistem Debian untuk menggunakan satu partisi tunggal untuk semua file sistem, yang mempermudah pengelolaan partisi.

  21. Setelah mengatur partisi, klik "Finish partitioning and write changes to disk". Ini akan menyelesaikan proses partisi dan menerapkan perubahan yang telah Anda buat pada disk.

  22. Setelah mengklik "Finish partitioning and write changes to disk", Anda akan diminta untuk mengonfirmasi. Klik "Yes" untuk menulis perubahan ke disk. Ini akan memulai proses penulisan partisi yang telah dikonfigurasi.

  23. Pada menu "Configure the package manager", pilih "No" ketika ditanya apakah Anda ingin memindai media instalasi tambahan. Ini akan mencegah pemindaian untuk media tambahan yang mungkin tidak diperlukan.

  24. Setelah memilih "No" pada konfigurasi media tambahan, Anda akan diminta untuk memilih negara untuk mirror paket. Pilih "Indonesia" untuk menggunakan server mirror lokal yang lebih dekat dengan lokasi Anda.

  25. Setelah memilih "Indonesia" sebagai negara untuk mirror, pilih "deb.debian.org" sebagai server mirror yang akan digunakan untuk mengunduh paket-paket Debian.

  26. Pada bagian "HTTP Proxy Information", biarkan kolom tersebut kosong dan lanjutkan dengan menekan "Continue". Ini menandakan bahwa Anda tidak menggunakan proxy untuk koneksi internet.

  27. Pada menu "Package Usage Survey", pilih "No" ketika diminta apakah Anda ingin berpartisipasi dalam survei penggunaan paket. Ini akan menonaktifkan pengiriman data anonim tentang penggunaan paket.

  28. Pada menu "Choose software to install", hilangkan tanda centang pada "Debian desktop environment" dan "... GNOME" dengan memilihnya dan menekan tombol Space. Ini akan memastikan bahwa lingkungan desktop dan GNOME tidak diinstal secara default, setelah itu tekan tombol enter untuk melanjutkan.

  29. Ketika diminta "Install the GRUB boot loader to your primary drive?", tekan "Yes". Ini akan menginstal GRUB sebagai boot loader utama pada drive Anda, sehingga Debian dapat di-boot setelah instalasi selesai.

  30. Setelah menyetujui instalasi GRUB, pilih "/dev/sda" atau hard disk utama tempat Anda ingin menginstal GRUB. Ini memastikan GRUB terpasang di drive utama sistem.

  31. Setelah GRUB berhasil diinstal, klik "Continue" untuk memulai proses reboot. Sistem Anda akan restart, dan Debian siap untuk digunakan setelah reboot.

Langkah 4: Login ke Debian

  1. Setelah sistem reboot, Anda akan diminta untuk login. Masukkan username yang Anda buat saat instalasi, misalnya kzmp, lalu tekan Enter.

  2. Setelah memasukkan username, masukkan password yang telah Anda buat untuk username tersebut, misalnya admin123. Perlu diingat, secara default password tidak akan terlihat saat diketik di Debian, jadi ketik dengan hati-hati, lalu tekan Enter.

  3. Selamat! Instalasi Debian Anda telah selesai dan sistem sudah siap digunakan. Anda kini dapat mulai menjelajahi Debian sesuai kebutuhan.
